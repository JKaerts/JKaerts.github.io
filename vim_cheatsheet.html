Vim cheatsheet

This is a living document of features that I want to remember or refer to at a future moment.

Splitting windows

:split Split window horizontally
:vsplit Split window vertically

You can navigate a split window using Ctrl-w followed by hjkl or the arrow keys. Closing the pane currently in focus is done with Ctrl-w followed by q.

Buffers

:buffers Show a list of all buffers
:ls Same as above
:b <number> Show the buffer with the given number.
:b <name> Show the buffer with the given (partial) name. Tab completion is available.
:bd <number> Delete the buffer with the given number (current buffer if absent). Fails if there are unsaved changes.
:bd! <number> Same as above but discards any unsaved changes.
